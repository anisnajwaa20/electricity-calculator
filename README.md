# Electricity Calculator

## Overview

This project is a simple Electricity Calculator developed using plain PHP and Bootstrap.

The system calculates:

- Power (kW)
- Energy consumption (kWh)
- Total electricity cost (RM)

based on user input for:

- Voltage (V)
- Current (A)
- Current electricity rate (sen/kWh)

The calculator automatically generates electricity usage and cost for 1 to 24 hours.

---

## Formula

Power (kW) = (Voltage × Current) / 1000
Energy (kWh) = Power × Hours
Total Cost (RM) = Energy × (Rate / 100)
